# 笔记管理系统

一个简洁、高效的基于浏览器的笔记管理应用，使用原生 HTML、CSS 和 JavaScript 开发。

## 功能特性

- **笔记管理**：创建、编辑、删除笔记
- **分类管理**：自定义笔记分类，支持编辑和删除分类
- **回收站**：删除的笔记会先进入回收站，可恢复或永久删除
- **搜索功能**：支持按标题或内容搜索笔记
- **深色主题**：一键切换浅色/深色主题
- **数据持久化**：使用 localStorage 本地存储数据
- **键盘快捷键**：支持常用操作的快捷键

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（包含响应式布局）
- **JavaScript ES6+** - 核心逻辑
- **Font Awesome** - 图标库

## 使用方法

### 启动应用

直接在浏览器中打开 `index.html` 文件即可使用，无需服务器环境。

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建笔记 |
| `Ctrl + S` | 保存笔记 |
| `Ctrl + F` | 聚焦搜索框 |
| `Ctrl + Delete` | 删除选中笔记 |

### 操作说明

1. **新建分类**：点击左侧分类栏的 `+` 按钮
2. **新建笔记**：点击中间列表栏的「新建笔记」按钮
3. **编辑笔记**：点击笔记列表中的任意笔记，在右侧编辑器中修改
4. **删除笔记**：选中笔记后点击「删除」按钮，笔记会移动到回收站
5. **恢复笔记**：在回收站中选中笔记，点击「恢复」按钮
6. **永久删除**：在回收站中选中笔记，点击「永久删除」按钮
7. **移动笔记**：选中笔记后点击「移动笔记」按钮，选择目标分类

## 数据存储

应用使用浏览器的 `localStorage` 存储数据，数据结构如下：

```javascript
{
  categories: [
    { id: 'cat_xxx', name: '分类名称', notes: ['note_id'], isDefault: false }
  ],
  notes: [
    { 
      id: 'note_xxx', 
      title: '笔记标题', 
      content: '笔记内容',
      createdAt: 'ISO日期',
      modifiedAt: 'ISO日期',
      categoryId: 'cat_xxx'
    }
  ]
}
```

**注意**：localStorage 的数据存储在浏览器本地，清除浏览器数据会导致笔记丢失。

## 响应式设计

- **桌面端**：三栏布局（分类栏 | 笔记列表 | 编辑器）
- **平板端**：编辑器宽度自适应
- **移动端**：垂直堆叠布局

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
